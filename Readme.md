# CareerTracker

A .NET application for tracking career progress and professional development.

## Getting Started

### Prerequisites
- .NET SDK

### Installation
```bash
git clone https://github.com/Md-Hasib-Askari/CareerTracker.git
cd CareerTracker
dotnet restore
dotnet build
```

### Usage
```bash
dotnet run
```
